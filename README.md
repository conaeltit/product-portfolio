## 🧭 Sobre mí

Desarrollo productos digitales como SPM con foco en medios de pago, construyendo desde cero el stack de productos de Virtualpos: una plataforma que hoy procesa pagos para más de 5.000 empresas en Chile.

Mi visión es facilitar la intersección entre producto, tecnología y regulación: desde definir la UX de un checkout hasta diseñar la arquitectura de estados de un sistema de suscripciones con dunning logic, integraciones 3DS y liquidación D+1. Lidero los productos hasta que son lanzados al mercado y realizo las intros y capacitaciones con clientes.

Me muevo bien entre discovery con usuarios, specs técnicas para engineers, y conversaciones de negocio con stakeholders.

## ⚡ Qué construí en VirtualPOS

| Producto | Descripción |
|---|---|
| **Web Checkout** | Checkout online con tarjetas, transferencias y PIX. 95% conversión. |
| **Quick Checkout** | Flujo sin fricción con tokenización de red para compras recurrentes. |
| **SmartPOS N86 Pro** | Terminal presencial con impresora, conexión SII y abono D+1. |
| **Suscripciones** | Pagos recurrentes online y presenciales con PAT. 97% conversión. |
| **Portal de Pagos** | Portal web integrable a ERPs, facturadores y bases de deuda. |
| **Link de Pago** | Cobro sin integración técnica ni tienda online. |
| **PIX** | Pagos QR para turistas brasileños. |
| **Integraciones & API** | Desarrollo de APIs y plugins paran conectar en minutos con plataformas externas propias o e-commerce. |
| **Reportes & Analítica** | Dashboards de conversión, riesgo y comportamiento. |
