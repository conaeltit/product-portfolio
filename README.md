# Product Portfolio

This repository contains the source code for CEltit fintech product portfolio website,
showcasing digital products built and scaled across Chile and Latin America.

## Purpose
- Demonstrate product strategy leadership
- Showcase fintech platforms, apps, and APIs
- Provide high-level technical and product insights

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

## Deployment
- Vercel
- Custom domain

## Governance
Product content reviewed for compliance and confidentiality.
